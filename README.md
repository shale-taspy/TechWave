# TechWave Podcast

TechWave is a modern, responsive podcast website designed to showcase a tech-focused audio content platform. It delivers daily insights on technology, productivity hacks, and personal growth stories from industry leaders.

## About

TechWave breaks down complex tech topics into digestible insights and explores productivity strategies that actually work. With over 150k monthly listeners, it's your go-to podcast for staying ahead in the fast-paced world of technology.

## Features

- **Responsive Design**: Fully responsive layout optimized for mobile, tablet, and desktop devices
- **Modern UI**: Clean and professional interface with gradient accents and smooth animations
- **Multi-Platform Integration**: Links to Spotify, Apple Podcasts, and YouTube
- **Social Media Links**: Connected to Instagram, LinkedIn, Twitter, and YouTube
- **Hero Section**: Eye-catching hero with animated microphone and hero graphics
- **Podcast Metrics**: Displays monthly listener statistics and engagement metrics
- **Call-to-Action Buttons**: Subscribe and Listen buttons with hover animations

## Tech Stack

- **HTML5**: Semantic markup structure
- **Tailwind CSS**: Utility-first CSS framework for styling
- **Font Awesome 7.0.1**: Icon library for social media and UI elements
- **Google Fonts**: Inter font family for typography
- **Responsive Design**: Mobile-first approach using Tailwind breakpoints

2. Open `index.html` in a web browser
3. Ensure all asset files are in the `assets/` folder
